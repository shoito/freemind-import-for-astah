FreeMind x Astah

FreeMind Import for astah

ソフトウェア設計支援ツール astah(旧JUDE) に、マインドマップ作成ツール FreeMind のファイル(.mm)をインポートするastahプラグインです。astah UML, professionalの何れかにインストールして利用します。

Demo

※このプラグインはastahのバージョン6.5.1以上で動作します。
※マインドマップを利用するため、astah communityでは利用できません。

Try it out!

FreeMind Import for astahは、astahにインストールするプラグインとしてデザインされています。

  1. 各OS用のastahインストーラーをダウンロードして、astahをインストールします。
    http://astah.change-vision.com/ja/product/astah-uml.html

  2. mm2asta-x.x.x.jarを astahインストールフォルダ/plugins に配置してください。アンインストールしたい場合は.jarファイルを削除してください。

    Mac OS X

    /Applications/astah UML/plugins

    Windows

    C:¥Program Files¥astah-UML¥plugins

Best Tools


Features

  • FreeMind(.mm)ファイルをastahに読み込める。
  • 複数のFreeMind(.mm)ファイルを一括でastahのマインドマップに変換できる。
  • 複数のマインドマップを一つのファイルで管理できる(astahの機能)。

Limitations

  • ノート/色/ユーザー定義アイコン/HTML/etcがインポートできない。
